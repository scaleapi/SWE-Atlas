```
Begin emission: 
Finished sending 1 packets.
Received 127 packets, got 0 answers, remaining 1 packets
```

The emission shows the packet was sent successfully, but got 0 answers - when sending ICMP to localhost, the kernel typically handles the loopback internally without routing through the network layer where Scapy's listener sits, so no reply is captured.

```
class IP(Packet, IPTools):
    fields_desc = [BitField("version", 4, 4),
                   BitField("ihl", None, 4),
                   XByteField("tos", 0),
                   ShortField("len", None),
                   ShortField("id", 1),
                   FlagsField("flags", 0, 3, ["MF", "DF", "evil"]),
                   BitField("frag", 0, 13),
                   ByteField("ttl", 64),
                   ByteEnumField("proto", 0, IP_PROTOS),
                   XShortField("chksum", None),
                   SourceIPField("src", "dst"),
                   DestIPField("dst", "127.0.0.1"),
                   PacketListField("options", [], IPOption, length_from=lambda p:p.ihl * 4 - 20)]
```
