I started investigating how DNS name compression works in Scapy by examining the core decompression function dns_get_str in layers/dns.py. When a DNS response arrives with names that fold back on themselves through compression pointers, the unwinding begins in this function, which gets called whenever the DNSStrField.getfield method parses a name from raw bytes.

The decompression algorithm works by walking through the byte string one byte at a time. Each byte is checked to see if it represents a normal label length or a compression pointer. Normal labels have their high two bits set to 00, meaning values from 0x00 to 0x3f represent label lengths. When the high two bits are 11 (values 0xc0 and above), this indicates a compression pointer where the remaining 14 bits encode an offset into the packet.
scapy/layers/dns.py
```python
while True:
    if abs(pointer) >= max_length:
        log_runtime.info(
            "DNS RR prematured end (ofs=%i, len=%i)", pointer, len(s)
        )
        break
    cur = orb(s[pointer])  # get pointer value
    pointer += 1  # make pointer go forward
    if cur & 0xc0:  # Label pointer
        if after_pointer is None:
            after_pointer = pointer + 1
        if pointer >= max_length:
            log_runtime.info(
                "DNS incomplete jump token at (ofs=%i)", pointer
            )
            break
        pointer = ((cur & ~0xc0) << 8) + orb(s[pointer]) - 12
```

The pointer calculation extracts the lower 6 bits of the first byte, shifts them left by 8, adds the full second byte, then subtracts 12. This subtraction accounts for the 12-byte DNS header (transaction ID, flags, and four count fields at 2 bytes each).

The format for encoding domain names uses length-prefixed labels. Each label starts with a byte indicating its length, followed by that many ASCII characters. A zero byte terminates the name. So "www.google.com." encodes as 0x03 "www" 0x06 "google" 0x03 "com" 0x00.

Tracking to prevent infinite loops when following compression pointers happens through the processed_pointers list in dns_get_str. Every time the parser follows a compression pointer, it appends that offset to this list. Before following any new pointer, it checks whether that offset already exists in the list.
scapy/layers/dns.py
```python
if pointer in processed_pointers:
    warning("DNS decompression loop detected")
    break
processed_pointers.append(pointer)
continue
```

When the check detects a loop, it emits a warning and breaks out of the parsing loop, returning whatever name fragments have been collected so far. This prevents denial-of-service attacks where a malicious packet creates circular references. The data "\x04test\xc0\x00" creates a name that says "test" and then points back to offset 0, which would restart the whole sequence. The parser correctly detects this, emits the warning, and returns "test." with whatever it collected before hitting the loop.

For out-of-bounds pointers that jump to locations that do not exist or land outside packet boundaries, the bounds check at the top of the loop catches this condition. When a pointer would jump to offset 255 but the packet is only 7 bytes, the abs(pointer) >= max_length check triggers, the info log records the premature end, and parsing stops gracefully.

The compression side of the machinery lives in the dns_compress function. When Scapy builds a DNS packet and compresses names to save space, it follows a specific pattern for finding compression opportunities.

scapy/layers/dns.py
```python
def possible_shortens(dat):
    """Iterates through all possible compression parts in a DNS string"""
    yield dat
    for x in range(1, dat.count(b".")):
        yield dat.split(b".", x)[x]
```

This function generates all possible suffixes of a domain name. The compression algorithm builds the packet first without compression, then scans through looking for duplicates. When it finds that a suffix like "example.com." already exists at a known offset, it replaces subsequent occurrences with a two-byte pointer.

scapy/layers/dns.py
```python
index = build_pkt.index(encoded)
fb_index = ((index >> 8) | 0xc0)
sb_index = index - (256 * (fb_index - 0xc0))
pointer = chb(fb_index) + chb(sb_index)
```

The first record keeps its full name "mail.example.com." The second and third records compress to just their unique prefix plus a pointer. The bytes "\x03www\xc0\x11" mean "www" (length 3) followed by a pointer to offset 0x11 (17 in decimal), where "example.com." begins in the first record.

When the same domain appears multiple times with different compression strategies, they all decompress to identical strings. This confirms that regardless of whether a name is stored compressed or uncompressed, the final decompressed value matches. The compression is purely a wire format optimization that becomes invisible after parsing.

The InheritOriginDNSStrPacket class plays a crucial role in decompression across record boundaries. When Scapy parses a DNS packet, it stores the original complete packet bytes in the _orig_s attribute.

scapy/layers/dns.py
```python
class InheritOriginDNSStrPacket(Packet):
    __slots__ = Packet.__slots__ + ["_orig_s", "_orig_p"]

    def __init__(self, _pkt=None, _orig_s=None, _orig_p=None, *args, **kwargs):
        self._orig_s = _orig_s
        self._orig_p = _orig_p
        Packet.__init__(self, _pkt=_pkt, *args, **kwargs)
```

In dns_get_str, when a compression pointer is encountered and the current byte string does not contain the target offset, the function switches to using pkt._orig_s if available.

def trace_decompress(s, pointer, label=""):
    max_length = len(s)
    name = b""
    after_pointer = None
    processed_pointers = []
    step = 0
    
    print(f"\n{label}Starting decompression at offset {pointer}")
    
    while True:
        step += 1
        if abs(pointer) >= max_length:
            print(f"  Step {step}: Pointer {pointer} >= max_length {max_length}, stopping")
            break
            
        cur = orb(s[pointer])
        print(f"  Step {step}: At offset {pointer}, byte=0x{cur:02x} ({cur})")
        
        pointer += 1
        
        if cur & 0xc0:
            if after_pointer is None:
                after_pointer = pointer + 1
                print(f"    -> Compression pointer detected (0xc0 bit set)")
                
            if pointer >= max_length:
                print(f"    -> Incomplete pointer, stopping")
                break
                
            next_byte = orb(s[pointer])
            raw_offset = ((cur & ~0xc0) << 8) + next_byte
            adjusted_offset = raw_offset - 12
            print(f"    -> Next byte: 0x{next_byte:02x}")
            print(f"    -> Raw offset: ((0x{cur:02x} & ~0xc0) << 8) + 0x{next_byte:02x} = {raw_offset}")

            pointer = adjusted_offset
            
            if pointer in processed_pointers:
                print(f"    -> LOOP DETECTED! Offset {pointer} already visited")
                break
                
            processed_pointers.append(pointer)
            print(f"    -> Jumping to offset {pointer}")
            continue
            
        elif cur > 0:
            label_bytes = s[pointer:pointer + cur]
            print(f"    -> Label length: {cur}")
            print(f"    -> Label: {label_bytes}")
            name += label_bytes + b"."
            pointer += cur
            print(f"    -> Accumulated name: {name}")
        else:
            print(f"    -> End of name marker (0x00)")
            break
    
    return name
